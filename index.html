📋 عن الكورس

ده كورس شامل وتفصيلي لبناء متجر إلكتروني كامل من الصفر. الكورس مصمم للناس اللي عايزين تتعلم البرمجة بالتطبيق العملي وتبني مشروع حقيقي تقدر تستخدمه أو تبيعه.

💡 ليه الكورس ده مختلف؟
  • تفصيلي - كل خطوة مشرح بالتفصيل حتى لو بديهية
  • عملي - هتبني مشروع حقيقي من الصفر مش مجرد نسخ كود
  • حديث - بيستخدم أحدث إصدارات التقنيات (Next.js 16, React 19)
  • شامل - من الأساسيات للنشر والبيع

📚 اللي هتتعلمه:

  • Next.js 16 - أحدث إصدار مع App Router و Server Components
  • React 19 - بناء واجهات المستخدم بأحدث الممارسات
  • Clerk Authentication - نظام تسجيل دخول احترافي بدون كود معقد
  • Supabase - قاعدة بيانات وتخزين سحابي مجاني
  • Tailwind CSS 4 - تصميم UI جميل وresponsive بسرعة
  • Framer Motion - حركات وتأثيرات احترافية
  • Admin Dashboard - لوحة تحكم كاملة لإدارة المتجر

🎯 مين الكورس ده مناسب ليه:

  • مطورين عايزين يتعلم Next.js بالتطبيق العملي
  • ناس عايزين تبني متجر إلكتروني حقيقي
  • مطورين عايزين يفهموا Clerk و Supabase
  • أي حد عايز يبني مشروع portfolio قوي
  • ناس عايزين تبيع قوالب جاهزة على منصات البيع

⏱️ مدة الكورس:

الكورس مكون من 25 درس موزعة على 7 وحدات. المدة التقديرية للإكمال هي 15-20 ساعة حسب سرعتك.

⚙️ المتطلبات

المتطلبات الأساسية:

  • معرفة أساسية بـ HTML, CSS, JavaScript (مش لازم محترف)
  • معلوميات أساسية عن React (مثلاً تعرف مكونات React إيه)
  • 计算机 عليه Node.js 18+ (هنثبتها مع بعض)
  • محرر كود (VS Code موصى به)

حسابات مجانية محتاج تعملها:

  • Supabase.com - حساب مجاني لقاعدة البيانات
  • Clerk.com - حساب مجاني لنظام المصادقة
  • Vercel.com - حساب مجاني للنشر
  • GitHub.com - حساب مجاني لحفظ الكود
✅ مفيش حاجة تانية محتاجها!
كل الحسابات دي مجانية وهتفيك للمشاريع الصغيرة والمتوسطة. محتاجش تدفع أي حاجة عشان تبدأ.
