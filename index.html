📋 عن الكورس

ده كورس شامل لبناء متجر إلكتروني كامل من الصفر. هتتعلم أحدث التقنيات في تطوير الويب وتبني مشروع حقيقي تقدر تبيعه أو تستخدمه كقالب لمشاريعك.

📚 اللي هتتعلمه:

  • Next.js 16 - أحدث إصدار مع App Router
  • Clerk Authentication - نظام تسجيل دخول احترافي
  • Supabase - قاعدة بيانات وتخزين سحابي
  • Tailwind CSS 4 - تصميم UI جميل وresponsive
  • Framer Motion - حركات وتأثيرات احترافية
  • Admin Dashboard - لوحة تحكم كاملة

🎯 مين الكورس ده مناسب ليه:

  • مطورين عايزين يتعلم Next.js بالتطبيق العملي
  • ناس عايزين تبني متجر إلكتروني حقيقي
  • مطورين عايزين يفهموا Clerk و Supabase
  • أي حد عايز يبني مشروع portfolio قوي

⚙️ المتطلبات

  • معرفة أساسية بـ HTML, CSS, JavaScript
  • معلوميات أساسية عن React
  • компьютер عليه Node.js 18+
  • حساب مجاني على Supabase.com
  • حساب مجاني على Clerk.com
