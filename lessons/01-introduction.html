📦 المشروع ده إيه؟

ده متجر إلكتروني كامل (E-Commerce) بنience من الصفر. المشروع بيشمل كل حاجة محتاجها متجر حقيقي:

  • 🏠 صفحة رئيسيةBeautiful مع Hero Section و Featured Products
  • 🛍️ صفحة المنتجات مع بحث وفلتر وتصفح
  • 📦 صفحة تفاصيل المنتج بالصور والمراجعات
  • 🛒 سلة مشتريات كاملة
  • ❤️ نظام مفضلة (Wishlist)
  • 🔐 نظام تسجيل دخول احترافي
  • 💳 صفحة Checkout
  • 📋 نظام طلبات
  • 📊 Admin Dashboard كامل

✨ مميزات المشروع

🎨 تصميم Glassmorphism

التصميم بيعتمد على تأثير Glassmorphism (الزجاج) اللي بيعطي شكل حديث وجميل للموقع. التصميم بيشمل:

  • خلفيات شفافة مع تأثير blur
  • ظلال ناعمة ومتدرجة
  • ألوان Indigo/Purple متناسقة
  • حركات سلسة مع Framer Motion

📱 Responsive Design

الموقع بيشتغل على كل الأجهزة:

  • موبايل 📱
  • تابلت 📱
  • لابتوب 💻
  • شاشات كبيرة 🖥️

⚡ أداء عالي

  • Server Components لتحميل أسرع
  • Image Optimization مع Next.js
  • Code Splitting تلقائي
  • Static Generation للصفحات الثابتة
