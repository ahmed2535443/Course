# 🎓 Market Course - كورس بناء متجر إلكتروني

كورس شامل وتفصيلي لبناء متجر إلكتروني كامل من الصفر باستخدام أحدث التقنيات.

## 📚 المحتوى

### الوحدة الأولى: الأساسيات (4 دروس)
- الدرس 01: مقدمة عن المشروع والتقنيات
- الدرس 02: إعداد بيئة التطوير
- الدرس 03: أساسيات Next.js 16 و App Router
- الدرس 04: إعداد Tailwind CSS 4

### الوحدة الثانية: قاعدة البيانات والمصادقة (3 دروس)
- الدرس 05: إعداد Supabase وقاعدة البيانات
- الدرس 06: إعداد Clerk Authentication
- الدرس 07: نظام Admin وحماية المسارات

### الوحدة الثالثة: واجهة المستخدم (5 دروس)
- الدرس 08: المكونات الأساسية
- الدرس 09: نظام التخطيط
- الدرس 10: الصفحة الرئيسية
- الدرس 11: صفحة المنتجات
- الدرس 12: صفحة تفاصيل المنتج

### الوحدة الرابعة: السلة والطلبات (4 دروس)
- الدرس 13: نظام السلة
- الدرس 14: صفحة Checkout
- الدرس 15: نظام الطلبات
- الدرس 16: نظام المفضلة

### الوحدة الخامسة: Admin Dashboard (3 دروس)
- الدرس 17: لوحة تحكم Admin
- الدرس 18: إدارة المنتجات
- الدرس 19: إدارة الطلبات

### الوحدة السادسة: التحسينات (2 درس)
- الدرس 20: Loading States
- الدرس 21: معالجة الأخطاء

### الوحدة السابعة: النشر والبيع (4 دروس)
- الدرس 22: SEO والأداء
- الدرس 23: النشر على Vercel
- الدرس 24: تخصيص القالب
- الدرس 25: دليل البيع

## 🛠️ التقنيات المستخدمة

- **Next.js 16** - إطار عمل React
- **React 19** - مكتبة واجهات المستخدم
- **Tailwind CSS 4** - إطار عمل CSS
- **Clerk** - نظام المصادقة
- **Supabase** - قاعدة البيانات
- **Framer Motion** - الحركات

## 🚀 التشغيل

1. حمّل الملفات
2. افتح `index.html` في المتصفح
3. ابدأ بالدرس الأول
